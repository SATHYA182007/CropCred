## 🧱 Tech Stack Requirements (Strict)

Use a modern, high-performance stack:

### Frontend
- React (use functional components)
- TypeScript (preferred over plain JS)
- Vite (for fast build and dev)
- Tailwind CSS (for styling)
- Framer Motion (for animations)
- Recharts (for graphs and data visualization)

### Backend
- Node.js
- Express.js API server
- RESTful API structure

### Data / State
- JSON-based mock data (for now)
- Proper folder structure for scalability
- Use React state management (Context API or Zustand)

### Project Structure
Generate a clean and scalable structure like:

- /src
  - /components
  - /pages
  - /layouts
  - /hooks
  - /utils
  - /data (JSON files)
  - /assets
- /server (Node + Express backend)
- /public
- package.json
- tsconfig.json

Ensure node_modules compatibility and proper dependency setup.
